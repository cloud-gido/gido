# 贡献者行为准则

## 我们的承诺

为了营造开放、友善的社区环境，我们承诺让每位参与者都能免受骚扰，无论年龄、体型、残障、 ethnicity、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向。

## 我们的标准

**鼓励的行为：**

- 使用友善、包容的语言  
- 尊重不同观点与经验  
- 优雅地接受建设性批评  
- 关注对社区最有利的事情  

**不可接受的行为：**

- 骚扰、侮辱/贬损评论、人身攻击或政治攻击  
- 公开或私下骚扰  
- 未经明确许可发布他人隐私信息  
- 其他在专业环境中可合理视为不当的行为  

## 执行

项目维护者可删除、编辑或拒绝不符合本准则的评论、提交、代码或 Issue，并在必要时暂时或永久禁止贡献者。

## 联系

通过 GitHub Issue 或 README 中的维护者联系方式报告不当行为。

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1。
