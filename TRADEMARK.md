> 本仓库**源代码**以 [Apache License 2.0](LICENSE) 授权。  
> **「玑渡」「GIDO」名称及官方视觉标识**不在 Apache-2.0 授权范围内，适用本政策。

---

## 1. 适用范围

| 资产 | 示例 |
|------|------|
| 中文名 | 玑渡 |
| 英文名 | GIDO |
| 组合名 | 玑渡 GIDO |
| 视觉 | 官方 Logo、星徽、登录页同款标识（见 `gido/frontend/public/brand/`） |
| Tagline | DATA · FLOW · VALUE、璇玑指引 · 数据有渡 |

---

## 2. 允许的使用（无需单独书面许可）

在**如实、非误导**的前提下，你可以：

1. 说明软件「基于 GIDO / 玑渡 GIDO 开源项目」或「Powered by GIDO」  
2. 在文档、README、关于页中引用项目名称以描述技术来源  
3. 在遵守 [LICENSE](LICENSE) 的前提下 fork 并分发**修改后的代码**  
4. 使用官方 Logo **仅用于**指向本仓库或官方发布页的链接（不得改形、不得作为你的产品主 Logo）

---

## 3. 禁止的使用

未经项目维护者**书面许可**，不得：

1. 将 fork 或衍生产品命名为「玑渡 GIDO 官方版」「Official GIDO」「GIDO Enterprise（暗示官方）」等  
2. 使用官方 Logo / 登录页视觉作为**你的产品品牌**，使用户误以为与维护者存在隶属、赞助或认证关系  
3. 在应用商店、域名、广告中使用「玑渡」「GIDO」作为**主品牌**销售商业产品（可与「Based on GIDO」并存说明技术来源，但主品牌须区分）  
4. 注册与「玑渡 GIDO」相同或近似的商标、域名用于同类大数据软件产品  

---

## 4. Fork 后的命名建议

| 场景 | 建议 |
|------|------|
| 个人学习 | 保留 GIDO 说明即可 |
| 企业内部部署 | `公司内部名 + based on GIDO` |
| 对外商业产品 | **自有产品名** + 开源致谢；替换 Logo 与登录页品牌区 |

视觉规范见 [gido/docs/BRAND.md](gido/docs/BRAND.md)。

## 5. 侵权处理

若发现未经授权使用官方商标或 Logo，维护者可能采取：

- 发送商标/版权说明函要求更正  
- 通过平台（GitHub 等）商标投诉流程处理  
- 在适用法律允许范围内追究法律责任  

**代码版权侵权**（未保留 LICENSE/NOTICE）可依据 Apache-2.0 及 DMCA 等机制处理。
